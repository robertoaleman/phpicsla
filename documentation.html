PHP Image Comparison with Simple Learning Algorithm v1.02
Author: Roberto Aleman
ventics.com
Code Explanation: This PHP code implements a basic image comparison system with machine learning. It allows users to upload two images, calculates a similarity percentage between them, and requests feedback to improve the calculation's accuracy.
Main Components:
  1. config.php:
    • Contains system configuration, such as the maximum allowed image size and allowed file types.
  2. comparacion_imagenes.php:
    • Presents a form for users to upload two images.
    • Validates uploaded images according to the configuration in config.php.
    • Displays the uploaded images side by side.
    • Uses the GD PHP extension to compare images and calculate a similarity percentage.
    • Presents a form for the user to indicate whether the similarity calculation is adequate.
    • Displays learning analysis results, including average similarities for similar and different images.
  3. feedback.php:
    • Receives user feedback on the similarity calculation.
    • Stores learning data in a CSV file named learning.csv.
    • Provides a link to return to the image comparison page.
  4. learning.csv:
    • Stores learning data, including image paths, calculated similarity percentage, and user feedback.
Key Functions:
  • compareImages():
    • Compares two images using the GD PHP extension.
    • Converts images to grayscale and calculates the color difference between pixels.
    • Returns a similarity percentage.
  • analyzeLearning():
    • Reads the learning.csv file and analyzes learning data.
    • Calculates average similarities for similar and different images.
    • Returns an array with analysis results.
  • validateImages():
    • Validates uploaded images according to the configuration in config.php.
    • Returns an array with found errors.
Usage Tutorial:
  1. Configuration:
    • Ensure you have PHP and the GD extension enabled on your web server.
    • Create a file named config.php and configure the $max_file_size and $allowed_types variables as needed.
  2. Files:
    • Save the img_compare.php and feedback.php files in the same directory as config.php.
    • Create a folder named uploads in the same directory.
    • Create an empty file named learning.csv in the same directory.
  3. Image Comparison:
    • Open the img_compare.php file in your web browser.
    • Select two images to compare and click "Compare".
    • Observe the images displayed side by side and the calculated similarity percentage.
  4. Feedback:
    • Answer the question "Is the similarity calculation adequate?" and click "Submit".
    • Observe the confirmation message and click "Go Back" to perform another comparison.
  5. Learning Analysis:
    • After performing several comparisons and providing feedback, observe the learning analysis results on the img_compare.php page.
    • The average similarities for similar and different images will give you an idea of how the system is learning.
